# Amazon E-Commerce Sales Performance & Trend Analysis

## 📌 Project Objective
The goal of this project is to process and analyze a massive e-commerce dataset containing over 51,000 transaction records to extract actionable business intelligence regarding revenue trends, regional profitability, and supply chain efficiency. 

## 📊 Key Insights & Findings
* **Revenue Mapping:** Executed Exploratory Data Analysis (EDA) to map out a total of $12.6M in overall revenue and $1.46M in net profit. 
* **Regional Performance:** Evaluated month-over-month (MoM) growth metrics, identifying the Central region as the highest performer, contributing $2.8M to total revenue.
* **Supply Chain Efficiency:** Analyzed shipping and fulfillment metrics to establish a 4-day average delivery time, setting a benchmark for optimizing operational expectations.
* **Loss Mitigation:** Successfully isolated high-volume but loss-making product categories to help guide future inventory and marketing strategies.

## 🛠️ Tools & Libraries Used
* **Language:** Python
* **Data Cleaning & Manipulation:** Pandas, NumPy
* **Data Visualization:** Matplotlib, Seaborn
* **Environment:** Jupyter Notebook

## 📂 Repository Contents
* `Amamzon_Sales_Analysis.ipynb`: The core Python notebook containing all data cleaning steps, EDA, and visual outputs.
* `SuperStoreOrders.csv`: The raw dataset used for analysis.

## ⚙️ How to Run the Project
1. Clone the repository: `git clone https://github.com/Saikiranpoluka/Amazon-Sales-EDA.git`
2. Install the required Python libraries:
   `pip install pandas numpy matplotlib seaborn jupyter`
3. Open the Jupyter Notebook:
   `jupyter notebook Amamzon_Sales_Analysis.ipynb`
